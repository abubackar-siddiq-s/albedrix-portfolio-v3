## Getting Started

Clone the repository:

```bash
git clone https://github.com/albedrixsystems/albedrix-portfolio-v3.git
```

Navigate to the project directory:

```bash
cd albedrix-portfolio-v3
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
